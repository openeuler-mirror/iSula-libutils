# iSula-libutils

#### Description

iSula-libutils provides log, json and common utils for iSula.


#### Installation

```
$ mkdir build
$ cd build/
$ cmake ../
$ make -j
$ sudo make install
```

LLT测试:

```
$ cd build/tests/
$ ctest
```

#### Instructions

1. provide library: libisula_libutils.so
2. provide header files: related to log and json
